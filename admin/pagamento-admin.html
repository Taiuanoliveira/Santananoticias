💳 Pagamento

Ver site →

Conecte sua conta do Mercado Pago para receber assinaturas de verdade (cartão, Pix, boleto). Depois de salvar aqui, o botão "Assinar" em Planos já leva o leitor direto para o checkout, sem precisar editar nenhum código.

Usado para pagamentos via Pix e Boleto. A pessoa paga manualmente e o acesso é liberado automaticamente.

Usado para cartão de crédito com cobrança automática todo mês. A pessoa assina uma vez e não precisa fazer mais nada.

Encontre em: painel de desenvolvedores do Mercado Pago → sua aplicação → Credenciais de produção → Access Token.

Como funciona

  1. O leitor clica em "Assinar" num plano em Planos.
  2. O site gera um link de pagamento seguro direto com o Mercado Pago.
  3. O leitor paga com cartão, Pix ou boleto, na própria página do Mercado Pago.
  4. Assim que o pagamento for aprovado, o acesso é liberado automaticamente — sem você precisar fazer nada.
  5. A assinatura expira sozinha na data certa, voltando o leitor para o plano gratuito.
