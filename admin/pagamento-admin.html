💳 Pagamento

Ver site →

Cole aqui os tokens do Mercado Pago. O sistema usa automaticamente o token certo dependendo do tipo de pagamento escolhido pelo leitor.

Para pagamentos via Pix e Boleto. A pessoa paga manualmente e o acesso é liberado automaticamente.

Para cartão de crédito com cobrança automática. A pessoa assina uma vez e renova sozinho.

Como funciona

  1. Cole os tokens acima e salve
  2. O leitor entra em Planos e escolhe Pix/Boleto ou Cartão
  3. É redirecionado pro Mercado Pago e paga
  4. O sistema libera o acesso automaticamente — sem você fazer nada
  5. A assinatura expira na data certa e o acesso é removido automaticamente
